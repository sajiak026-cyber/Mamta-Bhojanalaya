# Mamta-Bhojanalaya
📌 Mamta Bhojnalaya - Restaurant Website

A responsive restaurant website built using HTML, CSS, and JavaScript for Mamta Bhojnalaya.
It showcases menu items, pricing, and provides an interactive food ordering experience with a popup confirmation.

🌐 Live Features
Fully responsive design (mobile, tablet, desktop)
Attractive restaurant UI/UX
Menu with categorized food items
Interactive “Order Now” system
Popup confirmation message on order
Smooth navigation between sections
Clean and beginner-friendly code structure

🛠️ Tech Stack
HTML5
CSS3 (Flexbox + Grid + Media Queries)
JavaScript (DOM Manipulation)
